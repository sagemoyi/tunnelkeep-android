# CodeTunnel Android

A focused Android client for keeping a remote VS Code Tunnel workspace usable on a phone, especially across screen-off and transient network changes.

The project is being built for a real Android/HyperOS workflow. It is not affiliated with or endorsed by Microsoft, GitHub, OpenAI, or Anthropic.

## Target workspace

Default URL: `https://vscode.dev/tunnel/instance-20260610-18`

The URL will remain configurable so the app is useful to others and does not hard-code private credentials.

## Status

Initial implementation in progress.
